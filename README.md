## ✍️ Funcionalidades

- Login;
- Cadastro;
- Mostrar perfil;
- Atualizar perfil;
- Cadastrar filmes;
- Mostrar filmes cadastrados;
- Filtrar filmes;
- Mostrar detalhes do filme;
- Excluir filme;
- Logout.



## 🖥 Tecnologias

- JavaScript;
- Vite;
- React.js;
- Axios;
- AntD;
- React Icons;
- React Router.
